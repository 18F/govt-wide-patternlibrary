### **Standards outside linting**

We also carry the following working agreements that may fall outside of the linting.

---

#### **1. Avoid functionality in lithtml**

If you have this...

```js
html`<Foo @click={​​​​​​​​function () {​​​​​​​​ alert("1337") }​​​​​​​​}​​​​​​​​ />`
```

We prefer this

```js
html`<Foo @click={​​​​​​​​handleAlert}​​​​​​​​ />`
```

#### **2. Avoid inline styles**

If you have this...

```js
html`<Foo style={{ width: 100% }} />`
```

We prefer this

```js
html`<Foo class="full-width" />` />
```
